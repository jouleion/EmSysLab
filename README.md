# Files for ESL assignments

Several assignments have scripts and/or other useful files to get you started.

You can clone this repository using a git tool. This helps to keep the files up-to-date as there may be additions and updates throughout the course.

If you have no experience with git, then [Github Desktop](https://desktop.github.com/) or the built-in git manager in Visual Studio Code are good choices.

You can also download a zip file [here](https://git.ram.eemcs.utwente.nl/repository-esl/laboratory-files/-/archive/main/laboratory-files-main.zip?ref_type=heads).
